## About The Project

目標為快速建立DB Schema文件，並且利用Electron可包裝為應用程式，方便不熟悉NodeJS或非開發人員使用。

* 目前只支援PostgreSQL

### Built With
* [React](https://reactjs.org)
* [Nodejs](https://nodejs.org)
* [Electron](https://www.electronjs.org)



<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

安裝Nodejs
* npm
```sh
npm install npm@latest -g
```

目標資料庫的資料表建立相關資訊

### Installation

1. Clone the repo
```sh
git clone https://github.com/TCFeng/db-doc-generator
```
2. Install NPM packages
```sh
npm install
```

<!-- USAGE EXAMPLES -->
## Usage (不使用Electron啟動)
1. 啟動後台
```sh
npm start
```

2. 啟動Electron
```sh
npm electron-pack
```
